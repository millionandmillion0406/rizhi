## 🎯 核心功能

一站式 AI 影视解说 + 自动化剪辑工具：

```
视频素材 → AI理解画面 → 写文案 → TTS配音 → 字幕生成 → 自动剪辑 → 成片
```

### 主要能力

| 功能 | 说明 |
|:-----|:------|
| 🧠 **AI 理解视频** | 用多模态大模型看懂画面内容 |
| ✍️ **自动写文案** | LLM 根据画面生成解说词 |
| 🎙️ **TTS 配音** | 支持多种引擎（豆包、腾讯云、IndexTTS语音克隆、OmniVoice） |
| 📝 **字幕生成** | 自动生成+预览 |
| ✂️ **自动剪辑** | moviepy 自动剪辑成片 |
| 🎵 **AI 配乐** | Sonilo AI 自动配背景音乐 |
| 🎬 **导出剪映草稿** | 支持导出到剪映继续编辑 |
| 📱 **短剧混剪/解说** | 支持短剧类型的内容创作 |

## 🔧 技术栈

- **界面**: Streamlit（WebUI）
- **视频理解**: 多模态 LLM（Gemini/Qwen2-VL/TwelveLabs等）
- **文案**: LLM（支持硅基流动/DeepSeek等）
- **剪辑**: moviepy
- **TTS**: 豆包/腾讯云/IndexTTS-1.5/OmniVoice
- **部署**: Docker / 整合包 / uv本地运行
- **运行**: Python 3.12+
