## 🔴 五条铁规（工程纪律）

1. **改前先读** — 动手改服务器代码前，先 `cat` 全文读完
2. **改后验语法** — 每次修改后跑 `python -m py_compile`
3. **部署验存活** — 部署后 `curl` 确认 HTTP 200
4. **一次一处** — 一次只改一个地方，验完再改下一处
5. **每次留记录** — 改了什么、怎么改的、结果写进日志

**核心思维：一口气改三处一起部署 = 一起炸 = 不知道哪错了。**
