## 教训
1. Web服务部署后必须用真实浏览器测试，curl 不够
2. 一个功能加好测好再加下一个，不要一口气全改
3. 代码推 git 再改服务器，避免丢失原始版本
4. jmcomic 官方库已经覆盖了所有 JM 功能，反向代理/在线阅读是多余的
5. 远程改代码必先 cat 全文 → 改一处 → 编译验证 → 测试 → 记录
